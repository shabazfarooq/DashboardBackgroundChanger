DESCRIPTION
-----------
    Dashboard background changer is a Java based application designed to 
    allow Mac users to apply their own background image to the Dashboard.


INSTALLTION
-----------
    1) Simply place DBG.jar within the folder your wallpapers are located

    2) Run the application as root, this is necessary as the wallpaper will be
       copied to the Systems folder.

       java -jar DBG.jar

    3) Each wallpaper within the folder will be assigned a number.
       Simply select the number of the wallpaper you would like to use.

    4) Log out and log back in to apply the update.
